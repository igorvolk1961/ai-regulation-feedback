# Общественное обсуждение проекта ФЗ «О регулировании ИИ в РФ»

Репозиторий для сбора и координации предложений по проекту федерального закона  
**«Об основах государственного регулирования сфер применения технологий искусственного интеллекта в Российской Федерации»**.

🔗 **Официальная площадка обсуждения:** [regulation.gov.ru/projects/166424](https://regulation.gov.ru/projects/166424)  
📅 **Дедлайн подачи замечаний:** 15 апреля 2026 г.

---

## ⚠️ Зачем этот репозиторий?

Официальная платформа regulation.gov.ru требует входа через Госуслуги и имеет, на мой взгляд, не очень удобный интерфейс для совместной работы. 
Этот репозиторий — альтернативный канал для:
*   **Осторожных**, которые не хотят регистрироваться на госпортале;
*   **Экспертов**, которые хотят обсудить формулировки до отправки;
*   **Сообщества**, чтобы видеть позицию коллег.

На платформе обсуждения проекта regulation.gov.ru для обсуждения предлагаются следующие пункты/вопросы:

1. Оставьте своё мнение и/или прикрепите файл с предложениями.
2. Ваше общее мнение по предлагаемому регулированию
3. Существуют ли альтернативные (менее затратные и (или) более эффективные) способы решения проблемы?
4. Какие выгоды и преимущества могут возникнуть в случае принятия предлагаемого регулирования?
5. Какие риски и негативные последствия могут возникнуть в случае принятия предлагаемого регулирования?
6. Является ли предлагаемое регулирование оптимальным способом решения проблемы?
7. Повлияет ли принятие проекта НПА на расходные обязательства субъектов Российской Федерации?
8. Какое влияние на социально-экономическое развитие субъектов Российской Федерации может оказать предлагаемое регулирование?

По каждому из вопросов опросного листа можно написать свой ответ и/или приложить файл с ответом.

Файл volkov-submission-official.md со своими предложения по корректировке проекта я приложил к пункту 1.

В файле volkov-questionnaire.md.дал ответы на все остальные вопросы опросного листа.

Дополнительно к вопросу 5 (риски и негативные последствия) приложил файл volkov-technical-appendix.md
(На платформе названия всех файлов я написал на русском языке)

Если у кого есть мысли, как улучшить текст закона с технической точки зрения — кидайте в Issues или правьте файлы с моими предложениями напрямую.
Все конструктивные предложения из этого репозитория будут оформлены и отправлены на официальную площадку от имени координатора с указанием авторства, если таково будет желание автора, либо без него.

Дедлайн — 15 апреля. Если не выстрелит — на платформе обсуждения проекта закона останется позиция сообщества.
